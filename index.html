What this app does

It turns official hazard bulletins (PAGASA, PHIVOLCS, USGS, NDRRMC, LGU) into a simple, at-a-glance answer for the BGC/Taguig office: carry on, prepare, work from home, or suspend. A scan runs each hour and a full brief is prepared at noon. The latest result always shows on the Today tab.

Install on your phone

iPhone / iPad (Safari): open the app link → tap Share → Add to Home Screen.

Android (Chrome): open the app link → tap ⋮ → Install app.

Install on your computer

Chrome / Edge: open the app link → click Install in the address bar (or the button above) → Install.

Who receives this

The PH Business Continuity Admins group. Members can install the app and view every notification here any time. Alerts can be forwarded to Outlook or your team chat in one tap from each notification. Automated sending is planned for a future update.
